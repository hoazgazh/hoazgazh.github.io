# hoazgazh.github.io

Terminal-style portfolio website. Built with React, TypeScript and Styled-Components.

Based on [Sat Naing's terminal-portfolio](https://github.com/satnaing/terminal-portfolio).

## Commands

Type `help` in the terminal to see all available commands.

## Development

```bash
pnpm install
pnpm dev
```

## Deploy

Automatically deployed to GitHub Pages via GitHub Actions on push to `main`.
